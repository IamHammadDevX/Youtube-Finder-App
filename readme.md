# Youtube-Finder-App

A quota-safe YouTube search and filtering desktop application for Windows, built with Python and a GUI (PySimpleGUI or Tkinter), storing all data in CSV files.

## Features

- **YouTube Video Search:** Search for YouTube videos directly from the desktop app.
- **Filtering Tools:** Filter results based on specified criteria.
- **Quota-Safe:** Designed to minimize and manage YouTube API quota usage.
- **Desktop GUI:** Easy-to-use interface built with PySimpleGUI or Tkinter.
- **Offline Data Storage:** All search and filter data is saved locally in CSV files.

## Getting Started

### Prerequisites

- Python (version 3.7+ recommended)
- [PySimpleGUI](https://pysimplegui.readthedocs.io/en/latest/) or [Tkinter](https://docs.python.org/3/library/tkinter.html)

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/IamHammadDevX/Youtube-Finder-App.git
    cd Youtube-Finder-App
    ```
2. Install dependencies:
    ```bash
    pip install PySimpleGUI
    ```

## Usage

1. Run the main Python file:
    ```bash
    python main.py
    ```
2. Use the GUI to search for YouTube videos and filter results.
3. All data will be stored in CSV files in the project directory.
